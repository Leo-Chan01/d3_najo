Real Estate App

Sop APP featured Real Estate application project made with Flutter 2.5.1.0,it helps you speed up your workflow. This app includes 18 unique screens ready to integrate with your project (Firebase, PHP,Node.js ...) and a lot of reusable elements. One of the main perks of using this project is the possibility of changing colors and styles in just one click.

The same code is used for both iOS and Android.

Application Features :

    Requirements

  • Any Operating System (ie. MacOS X, Linux, Windows).

  • Any IDE with Flutter SDK installed (ie. IntelliJ, Android Studio, VSCode etc).

  • A little knowledge of Dart and Flutter.

    • Set Up an Editor


    Follow the steps below to add an editor plugin for Android Studio, IntelliJ, VS Code, or Emacs. If you want to use a different editor, that’s OK

    1- Android Studio and IntelliJ


    - Install Android Studio or IntelliJ

  • Android Studio, version 3.0 or later.

  • IntelliJ IDEA Community, version 2017.1 or later.

  • IntelliJ IDEA Ultimate, version 2017.1 or later.

    • - Install the Flutter and Dart plugins

    To install these:
  • Start Android Studio.

  • Open plugin preferences (Configure > Plugins as of v3.6.3.0 or later).

  • Select the Flutter plugin and click Install.

  • Click Yes when prompted to install the Dart plugin.

  • Click Restart when prompted.

    • 2- Visual Studio Code


    - Install VS Code

  • VS Code, latest stable version.

    • - Install the Flutter and Dart plugins

    To install these:
  • Start VS Code.

  • Invoke View > Command Palette…

  • Type “install”, and select Extensions: Install Extensions.

  • Type “flutter” in the extensions search field, select Flutter in the list, and click Install. This also installs the required Dart plugin.

  • Validate your setup with the Flutter Doctor command.(see picture below)





    • For more details visit :

    https://flutter.dev/docs/get-started/install

    Getting Started with UI Kit

    • After download and unzip the package, use preferred IDE (Android Studio / Visual Code / IntelliJ) to open the project Real EstateApp folder.

    • From the terminal: Run flutter pub get .
    • OR

    • From Android Studio/IntelliJ: Click Packages get in the action ribbon at the top of pubspec.yaml (see picture below).




    • From VS Code: Click Get Packages located in right side of the action ribbon at the top of pubspecs.yaml file.




    • Open the simulator to run iOS or Android

    • Then press the run button to start project

    • How to UI Kit?

    In the <Your Poject>/lib folder you should find 4 subdirectories:

    - components
    - models
    - pages
    - utils




    Let’s go through each of these subdirectories in more details.

    1- Components

    In this folder <Your Poject>/lib/components there are 16 Widgets ready to use in your App and some custom elements for the app ...etc.



    Just you can call the widget class and customize it with your parameters and you can get the stylish widget integrated in your app

    • After you making all you changes and customizations save the project.
    • Open Console, navigate to you project folder
    • Run flutter build apk --release you should get the apk file in the /output folder
    • Run flutter install to install you application on your connected devices
